# Den of Wolves: Infinite Domain (2026)

Megagame run on March 7, 2026 at West Acres, Fargo, ND.

## Published Pages

These are Jekyll pages that were (and still are) live on the site:

- **index.md** — Player hub with pre-game checklist, countdown timer, and links to all docs
- **rules-primer.md** — Single-page mechanics reference (turn structure, morale, trading, shuttles, combat). This was the "if you read one thing, read this" doc.
- **background-guide.md** — World-building lore: FTL expansion, Wolf War, the Attack, survivor fleet formation, old nations (FAS, CPA, SAN) and colonial nations (Proxima, Rosal, Gliese, Helvetii)

## Briefs

The `briefs/` directory contains 19 role-brief PDFs — one per ship/team. Filenames have random 4-character hex suffixes (e.g. `brief--aegis----3106.pdf`) so the URLs aren't guessable. These are public but unlinked; each team only got their own link via email.

The hex mapping is baked into `utils/generate-role-emails.py` (printed at the end of each run) and was also used in the email templates. If you need to look up which hex goes with which team, the filenames themselves are self-documenting.

Teams/ships covered:
- **ICS:** Aegis (flagship)
- **Council ships:** Star Alpha
- **FAS:** Dione
- **CPA:** Icebreaker, Fuerza León
- **SAN:** Lucas, Salvador
- **Rosal:** Shepherd, Endeavour
- **Proxima:** Quellon, Vulcan
- **Gliese:** Refinery 124
- **Helvetii:** Hephaestus, Baltar
- **INC (press):** Javelin
- **Wildcards:** Arbourist, Inquisitor, Firebrand

## Files Removed After the Game

These lived here during prep but were deleted before committing to keep player data out of git:

- `temp-assets/` — Working directory with `badge-briefs.yaml` (all role definitions, duties, nations), source docx files, cast list CSV, and any other raw prep material
- `email-template-dow-2026-role--*.html` — 19 role-assignment email templates (generated by `utils/generate-role-emails.py`, lived at project root)

## How It Was Built

1. Game design docs written in Google Docs, exported as `.docx`
2. `utils/convert-docx.py` converted to markdown
3. Role briefs assembled in `badge-briefs.yaml`, enriched with `utils/add-badge-fields.py` and cleaned with `utils/unwrap-briefs.py`
4. Brief PDFs created from the YAML content (manual/external process)
5. `utils/generate-role-emails.py` renamed briefs with hex suffixes and generated email templates
6. Templates filled in with player names/emails and sent out with team members CC'd together
